# Eventify - Система бронирования билетов на мероприятия

Eventify — это backend-приложение для управления мероприятиями и бронирования билетов. Разработано на Spring Boot с использованием JWT-аутентификации, ролевой модели (USER/ADMIN) и PostgreSQL.

## ⚙️ Функциональность

### Для пользователей (USER)
- ✅ Регистрация и авторизация (JWT)
- ✅ Просмотр списка мероприятий с пагинацией
- ✅ Просмотр деталей мероприятия
- ✅ Создание бронирования
- ✅ Просмотр своих бронирований
- ✅ Изменение количества билетов
- ✅ Отмена бронирования
- ✅ Настройка уведомлений

### Для администраторов (ADMIN)
- ✅ Создание мероприятий
- ✅ Редактирование мероприятий
- ✅ Удаление мероприятий
- ✅ Просмотр всех бронирований
- ✅ Подтверждение бронирований
- ✅ Отмена/удаление любых бронирований

### Дополнительно
- ✅ Автоматическая очистка просроченных бронирований (шедулер)
- ✅ Swagger/OpenAPI документация
- ✅ Actuator метрики
- ✅ Docker контейнеризация

## 📦 Требования

- Java 21
- Docker Desktop (для запуска PostgreSQL)
- Gradle 8.5+ (или использование встроенного wrapper)
- PostgreSQL 15 (или Docker)
